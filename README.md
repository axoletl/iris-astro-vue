# IRIS ASTRO VUE
- Astro: SSG, strict hydration and file based routing
- Vue: Component framework
- Barba.js: Prefetching, SPA-like router views and view transitions
- SASS: CSS preprocessor

## Prerequesites:
- Node 16 (I use v16.20.0)
- vue-cli (For linting)

## To run dev server
`npm run dev`

## To build
`npm run build`

## To lint
`npm run lint`
